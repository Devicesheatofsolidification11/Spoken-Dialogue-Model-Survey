## 🧩 Main Concepts

### 🎤 Speech Input

Speech input is the part of the system that listens to spoken language and turns it into usable data. In many models, this starts with audio features or speech recognition.

### 🧠 Intermediate Representation

The intermediate representation is the middle layer between speech input and speech output. It can hold meaning, intent, text, or another structured form that helps the model reason about the dialogue.

### 🗣️ Speech Output

Speech output is the part that turns the model result back into spoken language. It may use text-to-speech, speech synthesis, or another output method.

### 🔁 Generation Pattern

The generation pattern describes how the system builds its response. Some models work in stages. Others use end-to-end flow. The survey helps you see how these patterns differ.
